Задача 1

Почему первое равенство – неверно, а второе – верно?

        alert( [] == [] ); // false
        alert( [] == ![] ); // true

Какие преобразования происходят при вычислении?

Решение:

Первое неверно, потому что обьекты равны друг другу только если это один и тот же обьект

Второе равенство верно. Ко второму операнду применяется оператор ! - логическое НЕТ, которое преобразует его в false(массив - это обьект и равен true, !true == false). Первый операнд преобразуется в список элементов(у массива нет valueOf, сработает toString), в нашем случае в ''. А так как '' == false, равенство верно.
