Задача 1.

Каков будет результат выполнения этого кода?

        var value = 0;
        function f() {
            if (1) {
                value = true;
            } else {
                var value = false;
            }
            alert( value );
        }
        f();

Изменится ли внешняя переменная value ? Как изменится результат, если из строки

  var value = false

убрать ключевое слово var?

Решение:

В результате мы увидим true.

Внешняя переменная не изменится

Если убрать var результат останется тем же, но внешняя переменная изменится
