Задание:

            Напишите код, который выделит все ячейки в таблице по диагонали красным цветом
1:1 2:1 3:1 4:1 5:1
1:2 2:2 3:2 4:2 5:2
1:3 2:3 3:3 4:3 5:3
1:4 2:4 3:4 4:4 5:4
1:5 2:5 3:5 4:5 5:5 
        Решение:

            var quantity = 5; //количество td в tr
            //var quantity = document.getElementsByTagName('tr')[0].children.length; //альтернативный способ

            var tds = document.getElementsByTagName('td');
            for(var i = 0; i < tds.length; i+=quantity+1) {
              tds[i].style.backgroundColor = 'red';
            }
