Задание:

        Есть дерево из тегов ul и li.

        Напишите код, который для каждого элемента li выведет:

            Текст непосредственно в нём указанный (без подразделов).
            Количество всех вложенных в него элементов li.
  • Животные
    • Млекопитающие
      • Коровы
      • Ослы
      • Собаки
      • Тигры
    • Другие
      • Змеи
      • Птицы
      • Ящерицы
  • Рыбы
    • Аквариумные
      • Гуппи
      • Скалярии
    • Морские
      • Морская форель
    Решение:

        var lis = document.getElementsByTagName('li');
        for (var i = 0; i < lis.length; i++) {
          document.write('Text: ' + lis[i].innerText + ' br ');
          document.write('вложенных li = ' + lis[i].getElementsByTagName('li').length + 'br br');
        }
